# WeatherStation
ESP32 Weather Station

## WARNING
This is my first C++ program so good luck haha.

## USE
Copy Example_settings.h to settings.h and fill in your info

## Dependincies
All are in the arduino libraries.
  * APRS-IS
  * ArduinoJson
  * BetterOTA
  * ESP32Time
  * PubSubClient
  * SparkFunBME280
  * Wire
